Les avantages :

Parfaitement adapté pour le développement .NET, il possède toutes les fonctionnalités d’un éditeur de code ainsi que la possibilité de compiler, tester, lancer, publier et sauvegarder votre code dans un espace de stockage. L’une des fonctionnalités les plus importantes de Visual Studio est IntelliSense : un assistant de complétion de code qui vous accompagne en listant chaque ligne de code que vous produisez. Il analyse les membres, variables, mots clés et toute information ayant un lien avec ce que vous êtes en train d’écrire.

Voici un exemple d'intellisense, ici en tapant le début de la commande "background", intellisense, à la manière de Google, propose toutes les commandes possible.

Les inconvénients :

Visual Studio est un IDE puissant, ce qui peut parfois entraîner la lenteur de certaines fonctionnalités.
