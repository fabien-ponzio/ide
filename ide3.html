Atom

Acceuil

Sublime Text

Visual Studio Code

Les avantages :

Un éditeur codé en CoffeeScript facile à inspecter

Une intégration native de Node.js

la présence du préprocesseur CSS LESS offrant la possibilité de voir le résultat des modifications via un live reload intégré

un système de packages disponibles en open source sur GitHub pour le code

Atom excelle dans l'ajout de contenu additionnel, actuellement il y 6452 packs et thèmes additionnels disponibles !

Les inconvénients :

La seule version d’Atom disponible est OS X. Il faudra encore patienter pour Linux et Windows.

S'adresse plus particulièrement aux développeurs front office

Atom est téléchargeable ici !
